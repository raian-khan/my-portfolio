# 🌐 My Portfolio

This is my personal portfolio website built to showcase my projects, skills, and experience as a developer.

## 🚀 Live Demo
🔗 https://raian-khan.github.io/my-portfolio

## 🛠️ Tech Stack
- React (TypeScript)
- Tailwind CSS
- Framer Motion
- React Icons

## ✨ Features
- Responsive design for all devices
- Smooth animations and transitions
- Projects showcase with GitHub links
- Skills section with tech stack
- Downloadable CV
- Contact section

## 📂 Project Structure
- `src/components/sections` → Main sections (About, Projects, Skills, Contact)
- `src/components/layout` → Layout components (Navbar, Footer)
- `src/assets` → Images and static assets

## 📸 Projects Included
- House Price Prediction (ML)
- Tomato Disease Classification (CNN)
- Food Ordering System (PHP & MySQL)
- Portfolio Website

## 📦 Installation
```bash
git clone https://github.com/raian-khan/my-portfolio.git
cd my-portfolio
